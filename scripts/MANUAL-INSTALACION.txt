🖥️ REQUISITOS DEL SISTEMA
═══════════════════════════════════════════════════════════════════

🔧 SE INSTALAN AUTOMÁTICAMENTE:
   • Node.js v18 o superior
   • PHP v8.2 o superior  
   • Composer (gestor de dependencias)

💾 REQUISITOS MÍNIMOS:
   • Windows 10 o 11
   • 4 GB de RAM
   • 2 GB de espacio libre en disco
   • Conexión a Internet (solo para instalación inicial)

🌐 OPCIONAL:
   • Router/Red local (para compartir con otros dispositivos)

═══════════════════════════════════════════════════════════════════
🔒 CONFIGURACIÓN DE FIREWALL (Si hay problemas de acceso)
═══════════════════════════════════════════════════════════════════

Si otros dispositivos no pueden acceder, ejecutar como administrador:

1. 🖱️  Abrir "Símbolo del sistema" como administrador
2. 📝 Copiar y pegar este comando:

netsh advfirewall firewall add rule name="Lab Equipment System" dir=in action=allow protocol=TCP localport=8000

3. ⏎ Presionar Enter
4. ✅ Reiniciar el sistema

═══════════════════════════════════════════════════════════════════
🆘 SOLUCIÓN DE PROBLEMAS COMUNES
═══════════════════════════════════════════════════════════════════

❌ PROBLEMA: "No se puede acceder desde otro dispositivo"
✅ SOLUCIÓN: 
   • Verificar que ambos estén en la misma red WiFi
   • Configurar firewall (ver sección anterior)
   • Verificar que la IP sea correcta

❌ PROBLEMA: "Error al instalar dependencias"
✅ SOLUCIÓN:
   • Asegurarse de ejecutar como administrador
   • Verificar conexión a Internet
   • Ejecutar "scripts\instalar-dependencias.bat" manualmente

❌ PROBLEMA: "El navegador no se abre automáticamente"
✅ SOLUCIÓN:
   • Abrir manualmente: http://localhost:8000
   • Verificar que el sistema esté iniciado

❌ PROBLEMA: "Error de puerto 8000"
✅ SOLUCIÓN:
   • Ejecutar "scripts\detener-sistema.bat"
   • Esperar 30 segundos
   • Volver a iniciar el sistema
